# Lessons

This folder contains lesson-level material that expands the syllabus files into teachable units.

Each lesson follows the quality bar in [Lesson Standards](../LESSON-STANDARDS.md). Lessons should create concrete artifacts and should be grounded in durable engineering, security, reliability, privacy, and governance practices.
